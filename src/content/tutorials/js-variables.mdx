## JavaScript Variables

Variables are **named containers** for storing data values. You declare a variable once and can reuse it throughout your program.

JavaScript provides three keywords to declare variables:

| Keyword | Can Reassign? | Scope | Use When |
| :--- | :--- | :--- | :--- |
| `var` | ✅ Yes | Function | Old code (avoid in modern JS) |
| `let` | ✅ Yes | Block | Value will change |
| `const` | ❌ No | Block | Value will NOT change |

### Naming Rules for Variables

- Must start with a letter, `$`, or `_`
- Can contain letters, digits, `$`, and `_`
- **Case sensitive** — `name` and `Name` are different
- Cannot use reserved keywords like `let`, `const`, `function`

> **Good names:** `userName`, `totalPrice`, `isLoggedIn`
> **Bad names:** `x1`, `data2`, `temp`

> **Tip:** Use `const` by default. Switch to `let` only when you know the value needs to change.
