## CSS Colors

In CSS, colors can be specified in several ways: by **name**, **HEX code**, or **RGB value**.

### Color Value Types

| Type | Syntax | Example |
| :--- | :--- | :--- |
| Color Name | `color: red` | 140+ built-in names |
| HEX Code | `color: #FF5733` | `#RRGGBB` format |
| RGB | `color: rgb(255, 87, 51)` | Red, Green, Blue values 0–255 |
| RGBA | `color: rgba(255, 87, 51, 0.5)` | RGB + Opacity (0 to 1) |

> **Tip:** HEX codes are the most popular format in professional web development. Use tools like [Google's color picker](https://g.co/kgs/colorpicker) to find the perfect HEX value.
