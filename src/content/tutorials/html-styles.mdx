## The HTML Style Attribute

The HTML `style` attribute is used to add styles to an element, such as color, font, size, and more.

### Example - Try It!

<Editor 
    initialHtml={`<p>I am normal</p>\n<p style="color:red;">I am red</p>\n<p style="color:blue;">I am blue</p>\n<p style="font-size:50px;">I am big</p>`}
/>

### The HTML Style Attribute Structure

Setting the style of an HTML element, can be done with the `style` attribute.

The HTML `style` attribute has the following syntax:

`<tagname style="property:value;">`

The property is a CSS property. The value is a CSS value.

### Background Color

The CSS `background-color` property defines the background color for an HTML element.

### Example

<Editor 
    initialHtml={`<body style="background-color:powderblue;">\n\n<h1>This is a heading</h1>\n<p>This is a paragraph.</p>\n\n</body>`}
/>

### Text Color

The CSS `color` property defines the text color for an HTML element:

<Editor 
    initialHtml={`<h1 style="color:blue;">This is a heading</h1>\n<p style="color:red;">This is a paragraph.</p>`}
/>

### Fonts

The CSS `font-family` property defines the font to be used for an HTML element:

<Editor 
    initialHtml={`<h1 style="font-family:verdana;">This is a heading</h1>\n<p style="font-family:courier;">This is a paragraph.</p>`}
/>

### Text Size

The CSS `font-size` property defines the text size for an HTML element:

<Editor 
    initialHtml={`<h1 style="font-size:300%;">This is a heading</h1>\n<p style="font-size:160%;">This is a paragraph.</p>`}
/>

### Text Alignment

The CSS `text-align` property defines the horizontal text alignment for an HTML element:

<Editor 
    initialHtml={`<h1 style="text-align:center;">Centered Heading</h1>\n<p style="text-align:center;">Centered paragraph.</p>`}
/>
