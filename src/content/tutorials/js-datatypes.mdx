## JavaScript Data Types

JavaScript is a **dynamically typed** language — a variable can hold any type of value, and type can change at runtime.

The most common data types are:

| Type | Example | Description |
| :--- | :--- | :--- |
| `String` | `"Hello"` | Text in quotes |
| `Number` | `42`, `3.14` | Any numeric value |
| `Boolean` | `true`, `false` | Logical true or false |
| `Undefined` | `let x;` | Declared but no value assigned |
| `Null` | `null` | Intentionally empty value |
| `Object` | `{name: "Alice"}` | Key-value pairs |
| `Array` | `[1, 2, 3]` | Ordered list |
