## JavaScript Syntax

JavaScript syntax is the set of rules that define how JavaScript programs are written.

There are two types of values in JavaScript:
- **Literals** — Fixed values (e.g. `42`, `"Hello"`)
- **Variables** — Named containers that store values (e.g. `let x = 5;`)

### Syntax Quick Reference

| Concept | Syntax | Example |
| :--- | :--- | :--- |
| Declare with `let` | `let name = value;` | `let x = 5;` |
| Declare constant | `const name = value;` | `const PI = 3.14;` |
| String literal | `"text"` or `'text'` | `"Hello"` |
| Number literal | digits | `42` or `3.14` |
| Expression | value + operator | `x + y` |
| Comment | `// text` | `// this is a comment` |

> **Tip:** Use `const` when the value won't change, and `let` when it will. Avoid `var` in modern JavaScript.
