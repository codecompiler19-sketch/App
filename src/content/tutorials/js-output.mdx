## JavaScript Display Possibilities

JavaScript can "display" data in different ways:

- Writing into an HTML element, using `innerHTML`
- Writing into the HTML output using `document.write()`
- Writing into the browser console, using `console.log()`

The most common and useful way is using `innerHTML` to update content on the page.

### How Each Method Works

| Method | Where Output Appears | Best Used For |
| :--- | :--- | :--- |
| `innerHTML` | Inside an HTML element | Displaying results on the page |
| `document.write()` | Directly in the HTML stream | Quick testing only |
| `window.alert()` | A popup dialog box | Simple alerts (avoid in production) |
| `console.log()` | Browser DevTools Console | Debugging your code |

> **Tip:** `innerHTML` is the most practical method. Use it to update page content dynamically without reloading the page.
