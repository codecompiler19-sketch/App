## JavaScript Statements

A JavaScript **statement** is an instruction that tells the browser to do something. A JavaScript **program** is a list of these statements run top to bottom.

Statements are made up of: **Values**, **Operators**, **Expressions**, **Keywords**, and **Comments**.

### Key Rules

| Rule | Example |
| :--- | :--- |
| End statements with `;` | `let x = 5;` |
| Statements run top to bottom | First declared → first executed |
| White space is ignored | `let x=5;` = `let x = 5;` |
| Multiple statements per line | `let a=1; let b=2;` |

> **Tip:** Always end your statements with a semicolon. It keeps your code clear and avoids unexpected bugs.
