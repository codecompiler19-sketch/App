## HTML Lists

HTML lists allow web developers to group a set of related items in lists.

<Editor 
    initialHtml={`<h2>HTML Lists</h2>\n\n<h3>An Unordered HTML List</h3>\n<ul>\n  <li>Item</li>\n  <li>Item</li>\n  <li>Item</li>\n  <li>Item</li>\n</ul>\n\n<h3>An Ordered HTML List</h3>\n<ol>\n  <li>First item</li>\n  <li>Second item</li>\n  <li>Third item</li>\n  <li>Fourth item</li>\n</ol>`}
/>

### Unordered HTML List

An unordered list starts with the `<ul>` tag. Each list item starts with the `<li>` tag.

The list items will be marked with bullets (small black circles) by default:

<Editor 
    initialHtml={`<h2>Unordered List</h2>\n<ul>\n  <li>Coffee</li>\n  <li>Tea</li>\n  <li>Milk</li>\n</ul>`}
/>

### Ordered HTML List

An ordered list starts with the `<ol>` tag. Each list item starts with the `<li>` tag.

The list items will be marked with numbers by default:

<Editor 
    initialHtml={`<h2>Ordered List</h2>\n<ol>\n  <li>Coffee</li>\n  <li>Tea</li>\n  <li>Milk</li>\n</ol>`}
/>

### HTML Description Lists

HTML also supports description lists.

A description list is a list of terms, with a description of each term.

The `<dl>` tag defines the description list, the `<dt>` tag defines the term (name), and the `<dd>` tag describes each term:

<Editor 
    initialHtml={`<h2>Description List</h2>\n\n<dl>\n  <dt>Coffee</dt>\n  <dd>- black hot drink</dd>\n  <dt>Milk</dt>\n  <dd>- white cold drink</dd>\n</dl>`}
/>
