## CSS Selectors

A CSS **selector** tells the browser *which* HTML elements to apply styles to. There are three fundamental types.

### Selector Quick Reference

| Selector | Syntax | Targets |
| :--- | :--- | :--- |
| Element | `p { }` | All `<p>` elements |
| ID | `#myId { }` | The element with `id="myId"` |
| Class | `.myClass { }` | All elements with `class="myClass"` |
| Universal | `* { }` | Every element on the page |
| Group | `h1, h2, p { }` | All h1, h2, and p elements |

> **Tip:** IDs must be unique on a page (one element per ID). Classes can be shared across many elements — that's what makes them so powerful for reusable styles.
