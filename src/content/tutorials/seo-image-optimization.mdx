# Visuals that Perform: Image Optimization

Images play a dual role in SEO: they enhance the user experience and can drive significant traffic via Google Image Search. However, unoptimized images are the leading cause of slow websites.

## 1. Descriptive Filenames

Don't upload `IMG_1234.jpg`. Rename your files before uploading.

- **Bad**: `header-01.png`
- **Good**: `retro-gaming-controller-blue.png`

## 2. The Power of Alt Text

The `alt` attribute (Alternative Text) is used when an image can't be displayed and by screen readers. Search engines use it to understand the subject of the image.

```html
<img src="laptop.jpg" alt="A professional programmer working on a laptop in a bright office.">
```

## 3. Formats and Compression

Every kilobyte counts. Use modern image formats whenever possible.

- **WebP/AVIF**: Much smaller file sizes than JPEG or PNG with similar quality.
- **SVG**: For logos and icons. Infinite scalability with tiny file sizes.

## 4. Lazy Loading

Stop loading images that are near the bottom of the page until the user scrolls to them. 

```html
<img src="large-image.jpg" loading="lazy" alt="Example product image">
```
*Note: Don't lazy load your main "Hero" image at the top of the page!*

## 5. Responsive Images

Don't serve a massive desktop image to a small mobile phone. Use the `srcset` attribute.

```html
<img 
  src="img-800.jpg" 
  srcset="img-400.jpg 400w, img-800.jpg 800w" 
  sizes="(max-width: 600px) 480px, 800px" 
  alt="Responsive landscape image">
```

By following these steps, you satisfy both the "eyes" of your visitors and the "robotic brains" of search engines.
